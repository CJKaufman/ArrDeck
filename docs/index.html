Features

Everything you need.
Nothing you don't.

ArrDeck is built for power users who live inside their media stack. Every pixel is intentional. Every interaction is instant.

🎛️

Unified Dashboard

Service health, active downloads, upcoming releases and health alerts — all on one screen. No tab switching.

📊

Drag & Drop Grid

Rearrange and resize every dashboard widget in edit mode. Your layout, your rules. State persists across restarts.

⚡

Native Performance

Built on Tauri 2 using Edge WebView2. Uses ~35MB RAM at idle versus ~250MB for an Electron equivalent.

🎨

6 Custom Themes

Minimal, Swizzin, Autobrr, Kyle, Nightwalker and Napster — each with fully calibrated accent colors and surfaces.

📺

Full Library Management

Browse your Sonarr and Radarr libraries with grid and list views, bulk actions, sort, filter and live search.

🔔

Windows Notifications

Native toast notifications for completed downloads, new health warnings, and connection events — even when minimized.

under one roof.
